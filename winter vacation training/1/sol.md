## I - 三分法

#### 题目大意

#### 求多项式函数在[l,r]区间内的峰值对于的横坐标,精确度在1e-6内

#### 基本的三分模板题，将模板套进去即可,需要注意的是实数域三分

#### 三分模板
```
#define eps 1e-6//此处eps根据题意来定
double f(double x){
    double sum = 0;
    /* 此处为计算值的函数 */
    return sum;
}
double l,r;
double midl,midr;
while(l + eps <= r){
    midl = l + (r - l) / 3 // 靠近l的三等分点
    midr = r - (r - l) / 3 // 靠近r的三等分点
    if(f(midl) > f(midr))r = midr;
    else l = midl;
}
```
